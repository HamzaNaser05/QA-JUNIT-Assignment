# QA JUnit Assignment

Simple Java project for practicing unit testing with JUnit and Maven.

## Overview

- Includes core classes for `Calculator`, `Product`, `Recipe`, `RecipeBook`, and `UserService`.
- Contains JUnit test cases to verify behavior and validate functionality.

## How to run

```bash
mvn test
```

## Project structure

- `src/main/java` - application code
- `src/test/java` - test code
- `pom.xml` - Maven project configuration

## Notes

- This project is meant for learning JUnit testing
